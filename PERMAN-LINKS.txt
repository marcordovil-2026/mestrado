Usando Google Colab
Acesse Google Colab:

Vá para Google Colab.
Abrir o Notebook a partir do GitHub:

No menu superior, clique em File (Arquivo) -> Open notebook (Abrir notebook).
Clique na aba GitHub.
Digite o URL do GitHub ou apenas o nome do repositório marcordovil/PROJ-GIT e pressione Enter.
Selecione o notebook Proj001GH.ipynb da lista que aparecer.
Executar o Notebook:

Uma vez carregado, você pode executar cada célula do notebook clicando no ícone de play à esquerda de cada célula ou selecionando Runtime -> Run all (Executar tudo) no menu superior.
